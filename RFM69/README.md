### RFM69 Test Code

STM32CubeMX v5.0.1
Tried and tested on the STM32F303RE Nucleo Board.
The RFM69.c has some untested code, an attempt at creating compatibility to receive JeeLib packets.

Pinout  
_Nucleo - RFM69_  
PA9 - DIO0  
PB3 - SEL  
PB13 - CLK  
PB14 - MISO  
PB15 - MOSI

PA5 for the Onboard LED
